About & Methodology

How this dashboard works, what it can and cannot tell you, and where the data comes from.

What is this dashboard?

The India El Nino Monsoon Tracker is an independent, automated data dashboard that pulls numbers from official climate agencies every day and presents them in plain English. It is designed to help non-experts understand how the current El Nino event might affect India's southwest monsoon.

This is NOT an official forecast. It does not predict rainfall. The India Meteorological Department (IMD) at mausam.imd.gov.in is the only authority for official monsoon forecasts in India. This dashboard simply displays publicly available data and explains what it means historically.

The El Nino–Monsoon Connection

El Nino (warming of the central-eastern Pacific Ocean) is one of the strongest known influences on India's monsoon — but it is far from a perfect predictor:

  • Correlation: approximately −0.5 — meaning El Nino explains only about 25% of monsoon rainfall variability. Other factors (IOD, MJO, local conditions) matter a lot.
  • Only about half of El Nino years saw drought — since 1950, roughly 50-60% of El Nino years had below-normal monsoon rainfall. The rest were normal or above.
  • Impact varies by region — northwest India and parts of the Deccan are more vulnerable; northeast India and the Western Ghats are less affected.
  • IOD can offset El Nino — a positive Indian Ocean Dipole (IOD) in the same year often brings extra moisture to India, partially or fully cancelling El Nino's drying effect.

Blended El Nino Probability Model

Instead of showing a single agency's forecast, we blend multiple independent sources to get a more balanced probability estimate:

SourceWeightWhat it measures
IRI/Columbia University35%Multi-model ENSO forecast probabilities
NOAA Climate Prediction Center25%Official US ENSO outlook
Nino-3.4 SST Index20%Current sea surface temperature anomaly
Oceanic Nino Index (ONI)12%3-month running mean of Nino-3.4
Southern Oscillation Index (SOI)8%Atmospheric pressure difference (Tahiti−Darwin)

If a source is unreachable, its weight is redistributed proportionally among available sources. The confidence level reflects how many sources contributed.

Monsoon Stress Index (0–100)

This is a custom composite indicator — not an official scientific index. It combines:

  • Nino-3.4 SST anomaly (El Nino strength)
  • SOI (atmospheric coupling)
  • IOD/DMI (Indian Ocean state — positive IOD reduces stress)
  • Actual rainfall departure from normal (from IMD)
  • PDO (Pacific Decadal Oscillation — modulates El Nino impacts over decades)

The index is meant as a plain-English summary ("how worried should we be?") rather than a rigorous forecast tool. It is a historical tendency indicator, not a certainty.

Data Integrity Rules

  • No fabricated data. If a source cannot be reached, the tile shows "STALE" or "SOURCE UNREACHABLE" — never a guessed number.
  • Every figure carries attribution. Source name and "as of" timestamp are shown on each metric tile.
  • No invented forecasts. All interpretations are labelled as historical tendencies, not certainties.
  • Per-tile degradation. One broken source never blanks the whole dashboard — other tiles continue working independently.

Update Schedule

A GitHub Actions workflow runs daily at 4:00 AM IST (22:30 UTC). It fetches all sources, writes the results to latest.json, appends a snapshot to history.json, and pushes to GitHub Pages. If the workflow fails, an issue is automatically opened on the repository.

Limitations

  • This is a one-person project with automated scrapers — not a 24/7 monitored service.
  • Source websites change format without notice; scraping can break at any time.
  • The Monsoon Stress Index is a custom heuristic, not peer-reviewed science.
  • Regional rainfall data (subdivision-level) depends on IMD PDF bulletins which are fragile to parse.
  • Kharif sowing data from government sources has persistent accessibility issues.
